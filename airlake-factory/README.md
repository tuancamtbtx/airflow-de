# PyTrust
A shared python package support some GCP tasks
## Run Test Locally
**Must Have**
```
export PYTHONPATH=$PYTHONPATH:/pytrust
```
**Run Test**
```
make test
```
## Build Lib
```
make dist
```
## Usage This Lib
**Copy pytrust-1.0.0.tar.gz in dist folder to your project**

**Configure requirements.txt**
```
./packages/pytrust-1.0.0.tar.gz
```
